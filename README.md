# 328RFStamp

universelle Trägerplatine für ein CC1101 Funkmodul und einen ATmega328P in kleinstmöglicher Abmessung.<br/>
- ISP Anschluss erfolgt über die Pins des CC1101.
- FTDI Pins sind 2mm-RM!
